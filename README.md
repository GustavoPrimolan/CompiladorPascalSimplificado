Passo a passo para executar o programa:

1 - Abra o prompt de comando (WIN + R e digite CMD e tecle enter);
2 - Acesse o diretório do executável "main.exe" (exemplo:cd C:\Desktop);
3 - Abra o executável;
4 - Digite main.exe nomearquivo.pas

O programa identifica a extensão do programa, portanto é necessário que o programa esteja com a extensão .pas
